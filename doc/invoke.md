## exports(namespace, options, cb)

Receives a `namespace`, and an Hash of `options` to invoke a given
generator. The usual list of arguments can be set with `options.args`
(ex. nopt's argv.remain array)

### Params: 

* **String** *namespace* 

* **Object** *options* 

* **Function** *cb*
